NG Animated Slider is a very light weight and Awesome animated SVG Frame Slideshow with super smooth hardware accelerated transitions.It is a responsive and device friendly slider which works perfectly on all major devices.It has 7 different awesome animation layout.It is a very Powerful Professional HTML/CSS3 slider plugin, with Fullscreen background perfect to showcase your photography or general creativeness ideas in your website

== How to add this slider in Elementor Page Builder ==
1. Create a new Page and Add Text Editor or HTML elementor widget in the page
2. Add any of the shortcode you like in the widget
[slider layout="layout1"] //For slider Animation type1
[slider layout="layout2"] //For slider Animation type2
[slider layout="layout3"] //For slider Animation type3
[slider layout="layout4"] //For slider Animation type4
[slider layout="layout5"] //For slider Animation type5
[slider layout="layout6"] //For slider Animation type6
[slider layout="layout7"] //For slider Animation type7

Note: When you copy paste this shortcode [slider layout="layout7"] from here to your website, Please type " " before and after of layout7 and such by yourself.

== How to Work with the Plugin ==

1. Upload `ng-animated-slider.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WordPress Dashboard, and find Sliders post type and create new slider
4. Place [slider] shortcode in your pages
5. For Various slider Options, Go to Customizer/ NG Animated Sliders Options
